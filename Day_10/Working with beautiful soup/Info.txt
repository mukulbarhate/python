What is Web-Scraping?

ans:- The process of extracting data from websites is called as Web Scraping.

How does Web Scraping work?
ans:- 
1) load website
2) parse HTML data
3) Extract tidy data
4) Transform data into required format




Hacking vs Web Scraping
	in Hacking (which is not legal) you try to extract information from the backend database whereas in Web Scraping you extract information which is publicly available.
